# 🕷️ Página Web de Spider-Man

Una página web dedicada al Asombroso Hombre Araña, creada con HTML5 semántico y CSS moderno.

## 📖 Descripción

Este proyecto presenta una página web completa sobre Spider-Man (El Hombre Araña), incluyendo su historia, poderes, datos curiosos y una galería de imágenes. La página utiliza elementos HTML semánticos y un diseño responsivo moderno.

## 🗂️ Estructura del Proyecto

```
Projecto Parcial/
├── index.html          # Página principal con contenido de Spider-Man
├── styles.css          # Estilos CSS con diseño responsivo
├── images/             # Carpeta de imágenes
│   ├── fondo.jpg       # Imagen de fondo de la página
│   ├── vecteezy_spiderman-vector_73170.jpg
│   ├── vecteezy_spider-man-portrait-vector_14052991.jpg
│   └── vecteezy_a-man-wearing-superhero-suit-inspired-by-spiderman_25027801.jpg
└── README.md           # Documentación del proyecto
```

## 🎯 Características

### Estructura HTML Semántica
- ✅ `<article>` - Historia principal de Peter Parker
- ✅ `<section>` - Poderes y habilidades
- ✅ `<aside>` (x2) - Datos curiosos y películas principales
- ✅ `<footer>` - Información de derechos

### Diseño y Estilos
- ✅ **Diseño responsivo** - Se adapta a móviles y tablets
- ✅ **Imagen de fondo personalizada** - Con overlay para mejor legibilidad
- ✅ **Grid Layout** - Organización moderna del contenido
- ✅ **Efectos hover** - Interacciones elegantes
- ✅ **Colores temáticos** - Inspirados en Spider-Man (rojo, azul, negro)
- ✅ **Navegación suave** - Scroll automático entre secciones

### Contenido
- 📚 Historia de origen de Spider-Man
- 💪 Poderes principales (Fuerza, Sentido Arácnido, Lanzatelarañas)
- 🎬 Películas más importantes
- 📊 Datos curiosos del personaje
- 🖼️ Galería con imágenes vectoriales

## 🚀 Cómo usar

1. **Abrir la página**: Abre `index.html` en tu navegador web
2. **Navegar**: Usa el menú de navegación para moverte entre secciones
3. **Explorar**: Disfruta del contenido sobre Spider-Man

## 🎨 Personalización

Para personalizar la página:

- **Contenido**: Modifica el texto en `index.html`
- **Estilos**: Ajusta colores y diseño en `styles.css`
- **Imágenes**: Reemplaza las imágenes en la carpeta `images/`

## 📱 Compatibilidad

- ✅ Chrome, Firefox, Safari, Edge
- ✅ Dispositivos móviles y tablets
- ✅ Resoluciones desde 320px hasta 4K

## 👨‍💻 Tecnologías Utilizadas

- **HTML5** - Estructura semántica
- **CSS3** - Estilos modernos con Grid y Flexbox
- **Responsive Design** - Adaptable a cualquier dispositivo
